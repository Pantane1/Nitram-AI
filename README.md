## üöÄ Deployment on Vercel

To get this app running on Vercel, follow these exact steps:

1.  **Set Environment Variables**:
    *   In your Vercel Project Settings, go to **Environment Variables**.
    *   Add a new variable named `API_KEY`.
    *   Value: Your Google AI Studio API Key (get it from [aistudio.google.com](https://aistudio.google.com/)).

2.  **Permissions**:
    *   The app requires Microphone, Camera, and Geolocation access.
    *   Ensure you are using `https://` for these features to work in the browser.

3.  **Local Development**:
    *   If running locally, ensure your environment has `API_KEY` exported or set in a `.env` file.
